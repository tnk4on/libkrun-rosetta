# libkrun-rosetta

Rosetta 2 Support for libkrun: Technical Feasibility and Legal Analysis

📄 **Read the full report**: [https://tnk4on.github.io/libkrun-rosetta/](https://tnk4on.github.io/libkrun-rosetta/)

## About

This site hosts a technical investigation into adding Rosetta 2 (x86_64 binary translation) support to [libkrun](https://github.com/containers/libkrun), a lightweight VMM library used by [Podman](https://github.com/containers/podman) on macOS.

The report covers:
- A working prototype implementation and its results
- Apple's Rosetta protection mechanisms
- Legal analysis (DMCA, EULA, trademark, trade secret)
- How other OSS projects handle Rosetta
- Alternative approaches (FEX-Emu, Box64)
